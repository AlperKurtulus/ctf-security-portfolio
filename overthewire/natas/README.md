# Natas Wargame Overview

The Natas wargame is a web-based series of challenges designed to teach the fundamentals of web security. It consists of 34 levels, each progressively increasing in difficulty.

## Educational Disclaimer

This wargame is intended for educational purposes only. Players should only engage with the challenges in a safe, legal environment. It is not to be used for malicious purposes and should only be played if you have explicit permission.
